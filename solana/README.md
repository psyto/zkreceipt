# zktempo-verifier (Solana Anchor program)

Verifies Groth16 proofs of Tempo finality (generated by
[`zktempo-prover`](../prover/)) and persists the resulting
`(slot, state_root)` to a PDA. Downstream programs — e.g.
[`mppsol_cpi`](https://github.com/mppsol/cpi) — read the PDA + verify
Merkle Patricia inclusion proofs to authorize Solana-side settlement of
Tempo-originated payments.

## Important: nested workspace

This directory is a **nested Anchor workspace** — it is *not* a member of
the outer zkTempo Cargo workspace (the root `Cargo.toml` excludes it).
Build via Anchor's tooling, not the outer `cargo build`:

```bash
cd solana/
anchor build
```

**Toolchain requirements:**

- Anchor 0.32.1
- Solana CLI 3.1.14+ (ships platform-tools v1.52 with rustc 1.89 — supports
  `edition2024`, which recent transitive deps like `indexmap >= 2.14` require).
  Install via:
  ```bash
  agave-install init 3.1.14
  ```
  Older Solana CLI versions (2.2.x) bundle Cargo 1.79 and will fail to build
  with errors like `feature 'edition2024' is required`.

program ID into both `lib.rs` and `Anchor.toml`.
